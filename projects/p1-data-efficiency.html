Summary

The result in one minute

P1 extends P0 from model evaluation into experimental-design logic. P0 asks where zero-shot ESM scoring is reliable. P1 asks how many measured labels are needed before a supervised correction model can improve on that zero-shot score.

Around 384 measured labels, residual correction begins to reliably beat zero-shot scoring on real enzyme datasets. But the gain is not automatic at tiny label counts, and naive model-guided acquisition can underperform when it loses diversity.

Labels help, but label value depends on budget, selection strategy, mechanism region, and coverage of the mutation landscape.

01 / The connection

From P0 to P1

P0 produced the raw material: every variant carries an experimental fitness, a zero-shot ESM score, and residue-group labels. P1 asks what those experimental labels are worth.

01P0 scores mutations
Each variant gets experimental fitness, a zero-shot ESM score, and residue-group labels.
02P1 hides most labels
The benchmark simulates having only 24, 96, 384, or 1000 measured labels.
03P1 trains correction models
The model learns what the experiment teaches that zero-shot ESM missed.
04P1 evaluates acquisition
Different label-buying strategies are compared on held-out mutations.

P1 treats experimental labels as a finite campaign resource, not as an unlimited training set.

02 / Residual correction

Residual correction

prediction = zero-shot score + learned experimental correction
zero-shot score: the P0 ESM-2 prior, free before any assay learned correction: what measured labels add on top

The most important P1 model family starts with the ESM zero-shot score, then learns a correction from a small number of measured labels. This is stronger than throwing away the pretrained model, because it asks exactly what experimental data adds beyond zero-shot prediction.

05 / Learned representations

ESM delta embeddings help most when the question is local

True ESM delta embeddings do not radically change global performance. Their value shows up when the evaluation asks a biology-local question, inside mechanism-sensitive regions.

06 / Acquisition

Naive model-guided acquisition can lose to diversity

The agentic-looking strategies did not win. When the model only buys variants it already predicts to be best, it over-exploits narrow regions and loses the diversity it needs to keep learning.

07 / The scientific lesson

The scientific lesson

The lesson is not that active learning failed. The lesson is that naive acquisition is not campaign design. If the model only buys variants it already thinks are best, it can lose coverage of the landscape it needs to learn. In enzyme engineering, useful acquisition has to preserve diversity, cover mechanism-sensitive regions, and still exploit what the model has learned.

The next experiment is not simply the highest-scoring variant. It is the variant that teaches the campaign what it needs to know next.

Labels are a budget. The question is not how many you can run, but which ones are worth buying.

08 / What P1 proves

What P1 proves

Zero-shot is a useful prior

The pretrained model provides a real starting point before assay labels are available.

Labels become useful at sufficient budget

Supervised correction is weak at tiny label counts but becomes useful around 384 labels.

Embeddings help locally

True ESM embeddings produce modest global gains and stronger mechanism-slice gains.

Label selection matters

The value of a label depends on selection strategy, biological region, and landscape coverage.

Naive active learning can fail

Model-guided acquisition can underperform random or score-diverse sampling when it loses diversity.

09 / The boundary of the claim

What P1 does not prove yet

  • that 384 labels is universal across enzymes
  • a full real-data panel: AMIE and beta-glucosidase are not yet included
  • diversity-constrained model-guided acquisition
  • explicit uncertainty-aware acquisition
  • cost-per-label or plate-size-aware economics
  • a finished active-learning platform: this is a strong prototype

The project is strong because it exposes the next problem clearly: label acquisition has to be diversity-aware, mechanism-aware, and uncertainty-aware.

Zero-shot gives the first prior. Labels teach the correction. Acquisition decides whether the campaign buys information or only buys confidence.

P1 is a strong prototype: it shows that labels help once the budget is large enough, and that naive model-guided acquisition can lose to diversity. It is not a finished active-learning platform.
